CHAPTER 07  ·  모델 증류 의혹
DISTILLATION 큰 모델 → 작은 모델베끼기인가, 학습인가

"GPT의 답을 보고
— 나도 똑똑해졌다."

한 학생이 모범답안을 보고 공부하는 것처럼 — 작은 AI도 큰 AI의 답을 보고 학습할 수 있습니다. 이걸 모델 증류(distillation)라고 부릅니다.

스승 모델 · TEACHER
GPT-4 · 큰 모델
수만 시간·수억 달러로 학습. 어려운 문제도 답을 잘 냅니다.
→
학생 모델 · STUDENT
작은 모델
스승의 답 수백만 개를 보며 — 짧은 시간에 흉내합니다.
합법인 경우
자기 회사 큰 모델로 작은 모델을 만드는 것 — 정당한 압축입니다.
의심받는 경우
OpenAI 약관은 "GPT 출력으로 경쟁 모델 학습 금지"를 명시. 그래도 — 막을 방법이 없습니다.
2025 · OPENAI 주장
OpenAI는 "DeepSeek이 GPT 출력을 대량으로 빼서 증류했다는 정황을 봤다"고 발표. DeepSeek은 공식 부인. 증명은 — 사실상 불가능합니다.
왜 막기 어려운가
API 이용자가 계정 수십·수백 개로 분산해 질문하면 — 정상 사용과 구분되지 않습니다.
아이러니
OpenAI도 웹의 글·그림을 동의 없이 학습했다는 소송을 받고 있습니다 (NYT 등). — "누가 누구를 베꼈나"가 흐려지는 시대.
정리 · ESSENCE
증류는 기술적으로 합법이고 계약상 불법일 수 있습니다. 그 사이의 회색지대에서 — 후발 주자는 늘 앞선 자의 어깨에 올라탑니다.
