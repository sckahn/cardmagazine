자연어 · NATURAL LANGUAGE
“배가 고프다”
· 배(船)? 배(腹)? 배(梨)?
· 진짜 시장한가, 비유인가?
· 누구한테 말하는가?
· 문맥·감정·관계가 다 필요
→ 모호성 ↑ · 문맥 의존 ↑
코드 · CODE
if (x > 0)
  return x;
· if는 단 하나의 의미
· > 는 비교 연산자 단일
· return은 함수 종료·값반환
· 문맥 없음
→ BNF로 정의됨 · 모호성 0

코드는 BNF(Backus-Naur Form)와 문맥자유문법(CFG)으로 엄밀하게 정의됩니다. 모든 토큰의 의미가 사전에 못 박혀 있고, 같은 코드가 두 가지로 해석될 수 없습니다 — 컴파일러가 그렇게 만들어졌으니까요.

9호 LINGUA 콜백 · 촘스키의 위계
촘스키 위계에서 자연어는 가장 복잡한 0형(무제한)에 가깝고, 대부분의 코드는 2형(문맥자유)으로 처리됩니다. 사람의 말이 우주라면 — 코드는 한 평짜리 정원입니다.

AI에게 시는 어렵습니다. 코드는 그보다 훨씬 쉬운 문제입니다. 그래서 먼저 무너졌습니다.
